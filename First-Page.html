Tips and Tricks

  • CSS files use to style your pages.
  • JavaScript files can add on different interaction that's going to change the way people view and interact with your site.
  • Always Ctrl+S to Save your files.
  • Always use the validator to check your code.
    - HTML Validator - validator.w3.org
    - CSS Validator - jigsaw.w3.org/css-validator
  • Always test your code in as many browsers as you can (Microsoft Edge, Chrome, FireFox, Opera, etc.).
  • To nameing your files I recommended to use _(under scores) or -(hyphen or dashes) or capitalize the first letter of each word and no use spaces to sperating each words.
